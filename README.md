📏 ARuler — виртуальная линейка (AR)
Интерактивная виртуальная линейка с использованием дополненной реальности для измерения расстояний через камеру.
Поддерживает калибровку по эталонному объекту, отображение размера в реальных единицах и сохранение результатов.
Реализована на 7 языках программирования для демонстрации подходов к работе с видео, компьютерным зрением и интерактивными интерфейсами.

🧠 Концепция
ARuler — это виртуальная линейка, использующая камеру для измерения расстояний между двумя точками на объекте. Она позволяет:

✅ Отображать видео с камеры в реальном времени (в GUI-версиях).

✅ Калиброваться по эталонному объекту (например, по монете или кредитной карте) для пересчёта пикселей в миллиметры.

✅ Измерять расстояние между двумя кликами мыши на видео.

✅ Отображать линию и подпись с длиной прямо на видео.

✅ Сохранять результаты измерений в файл.

✅ Работать как в графическом режиме (с окном), так и в консольном (с вводом координат).

✅ Быть простым и интуитивно понятным.

🚀 Как запустить
Для GUI-версий требуется установка соответствующих библиотек. Консольные версии работают в интерактивном режиме.

Python
bash
pip install opencv-python numpy
python aruler_python.py
C++
bash
# Требуется OpenCV (sudo apt install libopencv-dev)
g++ -std=c++17 aruler_cpp.cpp -o aruler `pkg-config --cflags --libs opencv4`
./aruler
Java
bash
# Требуется JavaCV (скачать javacv.jar)
javac -cp .:javacv.jar aruler_java.java
java -cp .:javacv.jar aruler_java
C# (.NET Core)
bash
dotnet add package OpenCvSharp4.Windows
dotnet run
Go
bash
# Консольный режим (ввод координат)
go run aruler_go.go
Rust
bash
# Консольный режим
cargo run
JavaScript (Node.js)
bash
# Консольный режим
node aruler_js.js
